2006 葉氏盃，是我第一次以非賽員、非工作人員，卻幾乎全程參加的一次比賽。

這次來了許多各國的冠軍隊伍參加此次大賽，非常的精彩。


這是這次比賽的網頁。

賽程安排是所有的隊伍先打 12 Round 的 swiss 升降賽。（ 3 / 16 ~ 3 / 17 ）

而接著下來的就是分成兩組的 K.O. 淘汰賽。 (3 / 18 ~ 3 / 19 )

葉氏隊與 swiss 前七名為第一組，第八到第十五名則為第二組。

第一組被 K.O. 的隊伍，還有一次機會到第二組去做一次掙扎，而在第二組被淘汰只好與四萬美元的高額獎金訣別。

最後的第一組的冠軍，可以與第二組的冠軍一較高下，爭奪冠軍寶座。

在 swiss 賽失利的隊伍以及第一場 K.O. 賽被淘汰的隊伍則可以進入第三天的副盃試試看拿個安慰獎。 ( 3 / 18 )

最後一天有雙人賽，以及精彩的冠軍爭奪戰。 ( 3 / 19 )
